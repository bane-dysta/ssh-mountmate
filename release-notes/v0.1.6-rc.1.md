# SSH MountMate v0.1.6-rc.1

This is a pre-release for testing the latest Add/Edit config fixes before the next stable release.

## What's Changed

- Keep Save and Cancel fixed at the bottom of the Add/Edit config dialog, so form changes no longer push the buttons into the input area.
- Prevent the form from scrolling when its content already fits in the dialog.
- Fix the SSH config batch import panel appearing blank after the dialog button layout change.
- Keep mount-card action buttons in either one row of four or two rows of two; they no longer collapse into a tall single column.
- Add per-conflict actions for SSH config batch import:
  - new entries are imported as before;
  - duplicate entries are ignored by default;
  - matching saved configs can be manually overwritten.
- Expand SSH config batch import into a full import plan where every parsed Host can be reviewed, adjusted, and opened in a details window.
- Batch overwrite updates SSH connection fields while keeping local mount settings such as remote path and mountpoint.

## Notes

- This is a release candidate for user testing.
- `v0.1.5` remains the current stable release.

# SSH MountMate v0.1.6-rc.1 中文说明

这是一个用于测试的预发布版，主要验证新增/编辑配置页的最新修复。

## 主要更新

- 新增/编辑配置页的保存和取消按钮固定在底部，表单变化不会再把按钮挤到输入框区域。
- 内容高度足够时，表单不会再出现预期外的空白滚动。
- 修复按钮布局调整后 SSH config 批量导入页空白的问题。
- 主界面挂载卡片的四个操作按钮只会显示为一行四列或两行两列，不再压成过高的一列。
- SSH config 批量导入新增逐项冲突处理：
  - 新配置仍正常导入；
  - 重复配置默认忽略；
  - 能匹配到已保存配置的冲突项可以手动选择覆盖。
- SSH config 批量导入改为完整导入计划，每个解析到的 Host 都可以查看、调整动作并打开详情窗口。
- 批量覆盖只更新 SSH 连接字段，并保留远程路径、挂载点等本地挂载设置。

## 注意

- 这是供测试使用的候选版本。
- `v0.1.5` 仍是当前稳定版。
