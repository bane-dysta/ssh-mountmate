# SSH MountMate v0.1.4

This release keeps release packages cleaner while preserving bundled third-party license access.

## What's Changed

- Release zip packages now contain only the platform executable.
- Third-party notices and the bundled rclone MIT license are embedded in the app.
- Added `--licenses` to print bundled third-party notices from the command line.
- Added a Settings button to view bundled licenses in the GUI.
- The build script now clears `dist/` before building, preventing stale files from being packaged.

## Notes

- rclone is still bundled in release builds.
- The rclone MIT license is still retained and accessible in-app; it is not shipped as a separate file inside the release zip.
- Repository source files still include `THIRD_PARTY_NOTICES.md` and `licenses/rclone-COPYING.txt`.

# SSH MountMate v0.1.4 中文说明

这个版本让 release 包更干净，同时保留内置第三方许可证查看能力。

## 主要更新

- Release zip 包内现在只包含对应平台的可执行文件。
- 第三方声明和内置 rclone 的 MIT 许可证文本嵌入应用内。
- 新增 `--licenses`，可在命令行输出内置第三方声明。
- Settings 页面新增查看许可证按钮。
- 构建脚本会在构建前清空 `dist/`，避免旧文件被打进发布包。

## 注意

- Release 构建仍然内置 rclone。
- rclone MIT 许可证仍然保留并可在应用内查看；不再作为 release zip 内的单独文件分发。
- 仓库源码中仍保留 `THIRD_PARTY_NOTICES.md` 和 `licenses/rclone-COPYING.txt`。
