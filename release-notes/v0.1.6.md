# SSH MountMate v0.1.6

This release improves the Add/Edit config dialog, mount-card layout, and SSH config batch import workflow.

## What's Changed

- Keep Save and Cancel fixed at the bottom of the Add/Edit config dialog, so form changes no longer push the buttons into the input area.
- Prevent the Add/Edit config form from scrolling when its content already fits in the dialog.
- Fix the SSH config batch import panel appearing blank after the dialog button layout change.
- Keep mount-card action buttons in either one row of four or two rows of two; they no longer collapse into a tall single column.
- Expand SSH config batch import into a full import plan:
  - every parsed `Host` can be reviewed;
  - new entries can be imported or ignored;
  - duplicate entries are ignored by default;
  - matching saved configs can be manually overwritten;
  - each entry has a details window showing the parsed config, matched saved config, status, reason, and selected action.
- Batch overwrite updates SSH connection fields while keeping local mount settings such as remote path and mountpoint.

## Notes

- Release builds still bundle rclone.
- Windows still requires WinFsp separately for mounting.
- macOS still requires macFUSE separately for mounting.

# SSH MountMate v0.1.6 中文说明

这个版本改进了新增/编辑配置页、主界面挂载卡片布局，以及 SSH config 批量导入流程。

## 主要更新

- 新增/编辑配置页的保存和取消按钮固定在底部，表单变化不会再把按钮挤到输入框区域。
- 内容高度足够时，新增/编辑配置页不会再出现预期外的空白滚动。
- 修复按钮布局调整后 SSH config 批量导入页空白的问题。
- 主界面挂载卡片的四个操作按钮只会显示为一行四列或两行两列，不再压成过高的一列。
- SSH config 批量导入改为完整导入计划：
  - 每个解析到的 `Host` 都可以查看；
  - 新配置可以选择导入或忽略；
  - 重复配置默认忽略；
  - 能匹配到已保存配置的冲突项可以手动选择覆盖；
  - 每一项都有详情窗口，可查看解析出的配置、匹配到的已保存配置、状态、原因和当前动作。
- 批量覆盖只更新 SSH 连接字段，并保留远程路径、挂载点等本地挂载设置。

## 注意

- Release 构建仍内置 rclone。
- Windows 挂载仍需要单独安装 WinFsp。
- macOS 挂载仍需要单独安装 macFUSE。
